# Text Corpus

`text/chapters/` chứa corpus tiếng Việt release-candidate dùng để build EPUB.

- Tên file được zero-pad theo thứ tự split nguồn: `ch001-vn.txt` đến `ch126-vn.txt`.
- Dòng không trống đầu tiên của mỗi file được dùng làm heading chương và nhãn mục lục EPUB.
- Source tiếng Anh không được vendor trong repo này; dùng https://github.com/rrthomas/hpmor để đối chiếu nguồn.
