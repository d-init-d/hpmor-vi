# Workflow Artifacts

Các file này ghi lại workflow D Transcreate đã dùng cho bản release-candidate tiếng Việt của HPMOR.

| Phase | Artifact |
|---|---|
| Intake | `translation-brief.md` |
| Scan | `source-map.md` |
| Research | `glossary.md`, `glossary.csv`, `style-sheet.md`, `story-bible.md` |
| Plan | `context-plan.md`, `chunk-manifest.md` |
| Coordinate | `subagent-dispatch-plan.md` |
| QA | `qa-report.md`, `unresolved-issues.md` |

Repository cố ý loại bỏ backup cục bộ, corpus recovery thô, file split tạm và script sửa một lần. Các artefact được giữ lại là phần đủ bền để kiểm tra quy trình mà không mang theo rác từ máy làm việc.
